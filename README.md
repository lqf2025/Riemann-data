## Overview

This repository is organized to reproduce the main and extended figures listed above.

- `Fig3.py` generates **Fig. 3**
- `Fig4.py` generates **Fig. 4**
- `Ex_fig1.py` generates **Ex. Fig. 1**
- `Ex_fig2.py` generates **Ex. Fig. 2**

The `data/` directory contains the numerical input files used by the plotting scripts:

- `data.npz` and `data2.npz` are generated numerical datasets
- `data/exp/` contains processed experimental datasets
- `data/theory/` contains theoretical datasets used for comparison and plotting

In addition, `GLA.py` is used to generate the auxiliary files required by `Ex_fig1.py`:

- `GLA1.npz`
- `GLA2.npz`
- `GLA3.npz`

## About the `data/` directory

The `data/` directory contains both intermediate and prepared numerical datasets used in the figure scripts.

### Generated datasets

The files

- `data/data.npz`
- `data/data2.npz`

are generated numerical data files used by the plotting scripts. If needed, they can be regenerated by running:

```bash
cd data
python data.py
python data2.py
cd ..
```

### Experimental and theoretical datasets

The subdirectories

- `data/exp/`
- `data/theory/`

store the prepared datasets used in the comparisons and plots.

#### `data/exp/`

- `beta_scan_exp.npz`
- `tscan_beta03_exp.npz`
- `tscan_beta05_exp.npz`

#### `data/theory/`

- `beta_scan_theory.npz`
- `free_energy_axes_theory.npz`
- `tscan_beta03_theory.npz`
- `tscan_beta05_theory.npz`

## Reproducibility workflow

To reproduce all figures from scratch:

1. Install the required Python packages.
2. Make sure the `data/` directory is present.
3. If necessary, regenerate `data/data.npz` and `data/data2.npz` using:
   ```bash
   cd data
   python data.py
   python data2.py
   cd ..
   ```
4. Run the main plotting scripts:
   ```bash
   python Fig3.py
   python Fig4.py
   python Ex_fig2.py
   ```
5. For **Ex. Fig. 1**, first generate the GLA data if needed:
   ```bash
   python GLA.py
   python Ex_fig1.py
   ```

## Notes
